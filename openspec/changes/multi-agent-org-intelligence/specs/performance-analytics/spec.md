# 绩效目标 Agent 规格

## Overview

绩效目标 Agent 负责分析组织的绩效管理数据，包括 OKR/KPI 完成度、绩效评级分布、管理者风格识别等，为绩效体系优化提供洞察。

---

## 新增需求

### 需求: OKR 完成度分析

Agent 能够分析 OKR 的设定和完成情况。

#### 场景: 计算 OKR 完成率

- **当** 分析指定周期的 OKR 数据
- **则** 计算：
  - 整体 OKR 完成率
  - 按部门/团队的完成率分布
  - 按 O（目标）层级的完成率
  - 各 KR（关键结果）的完成情况
- **且** 识别完成率异常的部门/个人

#### 场景: OKR 质量评估

- **当** 分析 OKR 的设定质量
- **则** 评估：
  - 目标是否符合 SMART 原则
  - 挑战度是否合理（完成率过高/过低）
  - 与公司战略的对齐程度
  - KR 数量是否合理（3-5 个）
- **且** 生成 OKR 设定改进建议

#### 场景: OKR 进度追踪

- **当** 周期进行中
- **则** 预测 OKR 完成风险：
  - 根据当前进度预测最终完成率
  - 标记进度落后的目标
  - 识别可能无法完成的 KR
- **且** 生成进度预警报告

---

### 需求: 绩效分布分析

Agent 能够分析绩效评级的分布健康度。

#### 场景: 绩效等级分布

- **当** 分析绩效评级数据
- **则** 生成分布报告：
  - 全公司绩效等级分布（S/A/B/C/D）
  - 按部门的分布对比
  - 按职级的分布对比
  - 历史趋势对比
- **且** 识别分布异常（如某部门全 A）

#### 场景: 强制分布合规检查

- **当** 公司有强制分布要求
- **则** 检查各部门是否符合分布要求
- **且** 计算偏离程度
- **且** 标记不合规部门

#### 场景: 绩效通胀检测

- **当** 分析历史绩效数据
- **则** 识别绩效通胀迹象：
  - 平均绩效等级逐年上升
  - 低绩效比例逐年下降
  - 与业务结果脱节
- **且** 量化通胀程度

---

### 需求: 管理者风格分析

Agent 能够识别管理者的绩效管理风格。

#### 场景: 评分风格识别

- **当** 分析管理者的历史评分数据
- **则** 识别评分风格：
  - 宽松型（平均分显著高于公司平均）
  - 严格型（平均分显著低于公司平均）
  - 集中型（评分集中在某一等级）
  - 分散型（评分分布均匀）
- **且** 计算风格偏差程度

#### 场景: 管理有效性评估

- **当** 分析管理者下属的绩效表现
- **则** 评估管理有效性：
  - 下属绩效提升率
  - 下属离职率
  - 下属晋升率
  - 团队 OKR 完成率
- **且** 生成管理者画像

#### 场景: 问题管理者识别

- **当** 检测到以下模式
  - 评分极端（全 A 或全 C 以下）
  - 下属高离职率
  - 下属绩效持续下降
- **则** 标记为需关注的管理者
- **且** 生成约谈建议

---

### 需求: 绩效公平性分析

Agent 能够检测绩效评定中的潜在偏差。

#### 场景: 检测评分偏差

- **当** 分析绩效数据
- **则** 检测可能的偏差：
  - 性别偏差（男女绩效分布差异）
  - 年龄偏差
  - 司龄偏差
  - 部门偏差
- **且** 进行统计显著性检验
- **且** 生成公平性报告

---

### 需求: 绩效建议生成

Agent 能够基于分析生成改进建议。

#### 场景: 生成绩效体系优化建议

- **当** 完成绩效分析
- **则** 生成建议：
  - OKR 设定流程改进
  - 绩效校准机制优化
  - 管理者培训需求
  - 评估标准调整建议
- **且** 优先级排序
