# 招聘效能 Agent 规格

## Overview

招聘效能 Agent 负责分析招聘全流程数据，评估渠道效能、人岗匹配度、转化漏斗等，为招聘策略优化提供数据支撑。

---

## 新增需求

### 需求: 招聘渠道 ROI 分析

Agent 能够计算各招聘渠道的投入产出比。

#### 场景: 计算渠道成本效益

- **当** 分析指定时间段的招聘数据
- **则** 计算每个渠道的：
  - 投入成本（渠道费用、猎头费、广告费）
  - 简历数量
  - 有效简历数
  - 面试数
  - Offer 数
  - 入职数
  - 人均招聘成本（Cost Per Hire）
- **且** 按成本效益排序渠道
- **且** 标记低效渠道（成本高于阈值且入职率低于平均）

#### 场景: 渠道质量评估

- **当** 统计各渠道入职员工的后续表现
- **则** 计算：
  - 试用期通过率
  - 6 个月留存率
  - 12 个月留存率
  - 平均绩效评级
- **且** 生成渠道质量评分

---

### 需求: 招聘漏斗分析

Agent 能够分析招聘全流程的转化率。

#### 场景: 计算漏斗转化率

- **当** 分析招聘流程数据
- **则** 计算各环节转化率：
  - 简历筛选通过率
  - 初面通过率
  - 复面通过率
  - Offer 接受率
  - 入职报到率
- **且** 识别转化率异常环节（低于行业基准 20%）
- **且** 按部门、职级、职位拆解漏斗

#### 场景: 漏斗瓶颈诊断

- **当** 检测到某环节转化率显著低于平均
- **则** 分析可能原因：
  - JD 匹配度问题
  - 薪资竞争力问题
  - 面试体验问题
  - 流程效率问题
- **且** 生成优化建议

---

### 需求: 人岗匹配分析

Agent 能够评估招聘结果与岗位需求的匹配程度。

#### 场景: 技能匹配度评估

- **当** 分析入职员工的技能与岗位要求
- **则** 计算技能匹配度评分：
  - 硬技能匹配度
  - 软技能匹配度
  - 经验匹配度
  - 学历匹配度
- **且** 生成人岗匹配报告

#### 场景: 预测入职成功率

- **当** 收到新候选人信息
- **则** 基于历史数据预测：
  - 面试通过概率
  - Offer 接受概率
  - 试用期通过概率
- **且** 给出候选人优先级建议

---

### 需求: 招聘效率分析

Agent 能够评估招聘流程的时间效率。

#### 场景: 计算招聘周期

- **当** 分析招聘时间数据
- **则** 计算各维度的招聘周期：
  - 平均招聘周期（从需求提出到入职）
  - 各环节平均耗时
  - 按部门/职级的招聘周期分布
- **且** 识别超时职位
- **且** 对比行业基准

#### 场景: 招聘进度预警

- **当** 职位开放超过预设周期
- **则** 生成预警通知
- **且** 分析延迟原因
- **且** 建议加速措施

---

### 需求: 招聘建议生成

Agent 能够基于分析结果生成可执行建议。

#### 场景: 生成渠道优化建议

- **当** 完成渠道分析
- **则** 生成建议：
  - 应关闭的低效渠道
  - 应增加投入的高效渠道
  - 应尝试的新渠道
  - 预算重新分配方案
- **且** 估算优化后的预期收益
