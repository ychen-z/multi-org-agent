# 可视化看板规格

## Overview

React + ECharts 构建的管理看板，提供组织智能分析结果的可视化展示，支持实时交互、钻取分析和报告导出。

---

## 新增需求

### 需求: 总览仪表盘

提供组织核心指标的一览视图。

#### 场景: 核心指标展示

- **当** 进入看板首页
- **则** 展示核心指标卡片：
  - 员工总数 & 同比变化
  - 人效指标 & 目标达成率
  - 整体离职率 & 同期对比
  - 招聘完成率 & 进度
  - 组织健康度评分
- **且** 使用颜色编码表示状态（绿/黄/红）

#### 场景: 趋势图表

- **当** 查看趋势
- **则** 展示时间序列图表：
  - 人数变化趋势（12 个月）
  - 离职率趋势
  - 招聘进度趋势
  - 人效趋势
- **且** 支持时间范围筛选

---

### 需求: 招聘分析看板

展示招聘效能的详细分析。

#### 场景: 渠道 ROI 图表

- **当** 进入招聘分析页面
- **则** 展示：
  - 各渠道成本对比柱状图
  - 各渠道转化率漏斗图
  - 渠道 ROI 排名表
  - 渠道质量散点图（成本 vs 留存率）
- **且** 支持点击钻取到渠道详情

#### 场景: 招聘漏斗

- **当** 查看招聘漏斗
- **则** 展示交互式漏斗图：
  - 各环节人数和转化率
  - 与目标对比
  - 各环节平均耗时
- **且** 支持按部门/职位筛选

---

### 需求: 绩效分析看板

展示绩效相关的分析结果。

#### 场景: 绩效分布图

- **当** 进入绩效分析页面
- **则** 展示：
  - 全公司绩效等级饼图/环形图
  - 各部门绩效分布对比柱状图
  - 绩效分布历史趋势
  - 强制分布合规热力图
- **且** 支持切换维度（部门/职级/时间）

#### 场景: 管理者分析

- **当** 查看管理者维度
- **则** 展示：
  - 管理者评分风格雷达图
  - 管理有效性排名
  - 问题管理者标记列表
- **且** 支持点击查看管理者详情

---

### 需求: 人才风险看板

展示人才风险预警信息。

#### 场景: 风险热力图

- **当** 进入人才风险页面
- **则** 展示：
  - 部门离职风险热力图
  - 风险等级人数分布
  - 高风险人员名单（脱敏）
- **且** 使用颜色编码风险等级

#### 场景: 预警列表

- **当** 查看预警详情
- **则** 展示：
  - 高风险员工列表
  - 风险原因标签
  - 建议行动
  - 预警历史时间线
- **且** 支持标记已处理

---

### 需求: 组织健康看板

展示组织健康度评估。

#### 场景: 健康度仪表

- **当** 进入组织健康页面
- **则** 展示：
  - 组织健康度综合仪表盘
  - 各维度得分雷达图
  - 与历史/行业对比
- **且** 使用仪表盘组件展示得分

#### 场景: 人口结构图

- **当** 查看人口结构
- **则** 展示：
  - 年龄分布金字塔图
  - 司龄分布饼图
  - 学历分布柱状图
  - 性别比例图
- **且** 支持按部门筛选

---

### 需求: CEO 战略报告

生成并展示 CEO 一页纸报告。

#### 场景: 报告生成

- **当** 点击生成报告
- **则** 生成包含：
  - 执行摘要（关键数字）
  - 现状分析（核心指标 + 亮点/问题）
  - 风险预警（TOP 5 风险）
  - 策略建议（短/中/长期）
  - 可执行 Action List
- **且** 支持 PDF 导出

#### 场景: 报告定制

- **当** 配置报告选项
- **则** 支持：
  - 选择包含的模块
  - 调整时间范围
  - 选择对比基准
  - 添加备注
- **且** 保存报告模板

---

### 需求: 交互功能

看板支持丰富的交互操作。

#### 场景: 筛选器

- **当** 使用全局筛选器
- **则** 支持：
  - 部门筛选（多选）
  - 时间范围筛选
  - 职级筛选
  - 地域筛选
- **且** 所有图表联动刷新

#### 场景: 钻取分析

- **当** 点击图表元素
- **则** 支持：
  - 下钻到下一层级
  - 查看详细数据表
  - 跳转到相关分析
- **且** 支持面包屑导航返回

#### 场景: 数据导出

- **当** 导出数据
- **则** 支持：
  - 导出当前图表为图片
  - 导出数据为 Excel
  - 导出完整报告为 PDF
- **且** 保留筛选条件

---

### 需求: 响应式设计

看板适配不同设备。

#### 场景: 桌面端展示

- **当** 在桌面浏览器访问
- **则** 展示完整功能
- **且** 支持多图表并排展示
- **且** 支持快捷键操作

#### 场景: 大屏展示

- **当** 在数据大屏展示
- **则** 提供大屏模式
- **且** 自动轮播关键指标
- **且** 支持实时刷新
