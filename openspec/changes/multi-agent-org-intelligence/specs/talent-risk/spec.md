# 人才风险 Agent 规格

## Overview

人才风险 Agent 负责预测和预警人才相关风险，包括离职预测、高潜人才流失风险、团队稳定性评估等，帮助组织提前干预。

---

## 新增需求

### 需求: 离职风险预测

Agent 能够预测员工的离职概率。

#### 场景: 计算个人离职概率

- **当** 分析员工的多维度数据
- **则** 基于以下特征计算离职概率：
  - 司龄与职级匹配度
  - 薪资竞争力（与市场/内部对比）
  - 绩效评级趋势
  - 晋升间隔时长
  - 直属领导变更频率
  - 考勤异常（频繁请假、迟到）
  - 培训参与度下降
- **且** 输出离职概率评分（0-100）
- **且** 标记高风险员工（概率 > 70%）

#### 场景: 离职风险分层

- **当** 完成全员离职概率计算
- **则** 按风险等级分层：
  - 红色预警（> 80%）：需立即干预
  - 橙色预警（60-80%）：需重点关注
  - 黄色提醒（40-60%）：需定期沟通
  - 绿色安全（< 40%）：正常监控
- **且** 生成各层级人数统计

#### 场景: 离职原因归因

- **当** 识别高风险员工
- **则** 分析最可能的离职原因：
  - 薪资因素
  - 发展空间
  - 管理问题
  - 工作内容
  - 工作强度
- **且** 排序主要因素
- **且** 建议针对性挽留措施

### 需求: 高潜人才识别与保护

Agent 能够识别高潜人才并评估其流失风险。

#### 场景: 高潜人才识别

- **当** 分析员工综合数据
- **则** 基于以下维度识别高潜人才：
  - 绩效表现（连续 2 年 A 及以上）
  - 学习能力（培训表现、技能提升速度）
  - 领导力潜质（项目带领经验、360 反馈）
  - 适应性（跨部门/跨职能经验）
  - 创新贡献（专利、创新项目）
- **且** 计算高潜指数评分
- **且** 分类：超级高潜、高潜、潜力

#### 场景: 高潜流失预警

- **当** 高潜人才出现以下信号
  - 离职风险评分上升
  - 被猎头频繁接触（领英活跃度）
  - 重要汇报关系变化
  - 关键项目被调离
- **则** 生成紧急预警
- **且** 通知直属领导和 HRBP
- **且** 建议挽留方案

#### 场景: 关键岗位继任风险

- **当** 分析关键岗位
- **则** 评估继任风险：
  - 现任者离职风险
  - 内部继任者就绪度
  - 继任者数量（0/1/2+）
  - 继任者培养进度
- **且** 标记无继任者的高风险岗位

### 需求: 团队稳定性评估

Agent 能够评估团队的人员稳定性。

#### 场景: 团队流失风险评估

- **当** 分析团队维度数据
- **则** 计算团队稳定性指标：
  - 团队平均离职风险
  - 高风险人数占比
  - 近 12 月离职率
  - 预测未来 6 月离职人数
- **且** 按部门排序风险等级

#### 场景: 连锁离职风险识别

- **当** 关键人员（团队核心、技术骨干）离职风险高
- **则** 分析可能的连锁影响：
  - 强依赖关系的下属
  - 紧密合作的同事
  - 项目关键节点人员
- **且** 预测连锁离职人数
- **且** 生成团队稳定性加固建议

---

### 需求: 风险预警通知

Agent 能够自动发送风险预警。

#### 场景: 实时预警推送

- **当** 检测到风险等级变化
- **则** 按紧急程度推送通知：
  - 红色预警 → 即时通知 HRBP + 直属领导
  - 橙色预警 → 日报汇总
  - 黄色提醒 → 周报汇总
- **且** 记录预警历史

#### 场景: 生成风险周报

- **当** 每周固定时间
- **则** 生成人才风险周报：
  - 本周新增高风险人员
  - 风险等级变化人员
  - 已离职人员回溯分析（预测准确率）
  - 下周重点关注人员
