# 主控 Agent 规格

## Overview

主控 Agent（Orchestrator）是整个系统的总指挥，负责调度各专业 Agent、整合分析结果、进行交叉归因、生成最终战略报告和可执行 Action List。

---

## 新增需求

### 需求: Agent 调度

主控 Agent 能够协调各专业 Agent 的工作。

#### 场景: 任务分发

- **当** 收到分析请求（全面分析/专项分析）
- **则** 解析请求意图
- **且** 确定需要调用的 Agent 列表
- **且** 按依赖关系排序（数据治理 → 专业分析）
- **且** 分发任务到各 Agent

#### 场景: 并行执行

- **当** 多个 Agent 可并行执行（无依赖关系）
- **则** 并行调用这些 Agent
- **且** 收集各 Agent 返回结果
- **且** 处理执行异常（超时、失败）

#### 场景: 结果汇总

- **当** 所有 Agent 完成执行
- **则** 汇总各 Agent 的分析结果
- **且** 验证结果完整性
- **且** 进入交叉归因阶段

---

### 需求: 交叉归因分析

主控 Agent 能够发现不同维度数据之间的关联。

#### 场景: 招聘-绩效关联分析

- **当** 整合招聘和绩效数据
- **则** 分析：
  - 各渠道入职员工的绩效分布
  - 人岗匹配度与绩效的相关性
  - 招聘周期与绩效的关系
- **且** 生成渠道质量排名

#### 场景: 绩效-离职关联分析

- **当** 整合绩效和离职数据
- **则** 分析：
  - 高绩效员工的离职风险
  - 绩效变化与离职概率的关系
  - 绩效评定公平性与离职率的关系
- **且** 识别"高绩效高风险"群体

#### 场景: 管理者-团队关联分析

- **当** 整合管理者风格和团队数据
- **则** 分析：
  - 管理风格与团队绩效的关系
  - 管理风格与团队离职率的关系
  - 管理风格与敬业度的关系
- **且** 识别问题管理者

#### 场景: 组织健康-业务关联分析

- **当** 整合组织健康和业务数据
- **则** 分析：
  - 人效与组织结构的关系
  - 编制合理性与业务增长的关系
  - 人口结构与创新能力的关系
- **且** 生成组织优化方向

---

### 需求: 战略报告生成

主控 Agent 能够生成 CEO 一页纸战略报告。

#### 场景: 生成现状摘要

- **当** 完成全面分析
- **则** 生成现状部分：
  - 核心指标仪表盘（人效、人数、成本）
  - 组织健康度评分
  - 关键发现（3-5 条最重要洞察）
- **且** 使用数据可视化展示

#### 场景: 生成风险预警

- **当** 完成风险分析
- **则** 生成风险部分：
  - 高风险部门清单（排序）
  - 核心人才流失预警（具体到人）
  - 组织结构风险点
  - 风险量化（可能损失金额）
- **且** 按紧急程度排序

#### 场景: 生成策略建议

- **当** 完成归因分析
- **则** 生成策略部分：
  - 短期行动（1-3 个月）
  - 中期规划（3-6 个月）
  - 长期战略（6-12 个月）
  - 预期收益估算
- **且** 与公司战略目标对齐

---

### 需求: Action List 生成

主控 Agent 能够生成可执行的行动清单。

#### 场景: 招聘优化 Action

- **当** 基于招聘分析结果
- **则** 生成具体行动：
  - 关闭哪个低效招聘渠道（具体渠道名）
  - 增加哪个渠道投入（具体金额建议）
  - 优化哪些 JD
  - 改进哪个环节流程
- **且** 预估每个行动的收益

#### 场景: 人才挽留 Action

- **当** 基于离职风险分析
- **则** 生成具体行动：
  - 约谈哪些高风险员工（具体名单）
  - 调整哪些人的薪酬（范围建议）
  - 安排哪些人的发展计划
  - 调整哪些汇报关系
- **且** 排优先级

#### 场景: 管理改进 Action

- **当** 基于管理者分析
- **则** 生成具体行动：
  - 约谈哪几位问题管理者
  - 安排哪些管理者培训
  - 调整哪些管理幅度
  - 考虑哪些管理者调岗
- **且** 提供谈话要点建议

#### 场景: 薪酬调整 Action

- **当** 基于薪酬竞争力分析
- **则** 生成具体行动：
  - 哪类岗位需紧急调整薪酬带宽
  - 具体调整幅度建议
  - 预算影响估算
  - 优先级排序
- **且** 考虑内部公平性

---

### 需求: 对话交互

主控 Agent 支持与用户的自然语言对话。

#### 场景: 问题解答

- **当** 用户提问（如"为什么 A 部门离职率高？"）
- **则** 调用相关 Agent 获取数据
- **且** 组织回答
- **且** 提供数据支撑
- **且** 建议后续问题

#### 场景: 深度钻取

- **当** 用户要求深入分析某个点
- **则** 调用专业 Agent 做详细分析
- **且** 返回多维度拆解
- **且** 可视化展示

#### 场景: 假设分析

- **当** 用户提出假设场景（如"如果给 A 部门加薪 10%"）
- **则** 模拟场景影响
- **且** 预测指标变化
- **且** 提供决策建议
