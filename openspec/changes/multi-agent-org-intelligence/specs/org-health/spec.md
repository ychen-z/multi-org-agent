# 组织健康 Agent 规格

## Overview

组织健康 Agent 负责从宏观视角评估组织的整体健康度，包括人效分析、编制合理性、人口结构、敬业度等维度。

---

## 新增需求

### 需求: 人效分析

Agent 能够计算和评估组织人效。

#### 场景: 计算人效指标

- **当** 分析组织业绩和人员数据
- **则** 计算人效指标：
  - 人均营收（Revenue per Employee）
  - 人均利润（Profit per Employee）
  - 人均产出（Output per Employee）
  - 人工成本占比（Labor Cost Ratio）
- **且** 按部门/业务线拆解

#### 场景: 人效对标分析

- **当** 计算完人效指标
- **则** 进行对标分析：
  - 与历史同期对比
  - 与公司目标对比
  - 与行业基准对比（如有数据）
- **且** 识别人效异常的部门
- **且** 分析人效变化原因

#### 场景: 人效预测

- **当** 基于历史趋势
- **则** 预测：
  - 下一季度人效
  - 达成目标所需人效提升幅度
  - 人效提升的可行路径
- **且** 生成人效改进建议

---

### 需求: 编制分析

Agent 能够评估组织编制的合理性。

#### 场景: 编制使用率分析

- **当** 分析编制和在编数据
- **则** 计算：
  - 编制使用率（在编/编制）
  - 空缺岗位数
  - 超编岗位数
  - 按部门/职能的编制分布
- **且** 识别编制异常（使用率 < 70% 或 > 100%）

#### 场景: 编制合理性评估

- **当** 分析业务量和人员配置
- **则** 评估编制合理性：
  - 管理幅度（每个管理者直接下属数）
  - 职能比例（支持职能/业务职能）
  - 层级深度（汇报层级数）
  - 冗余度评估
- **且** 与行业最佳实践对比

#### 场景: 编制预测与建议

- **当** 基于业务规划
- **则** 预测所需编制：
  - 明年业务增长所需新增编制
  - 效率提升可优化的编制
  - 结构调整建议（增哪里减哪里）
- **且** 生成人头预算建议

---

### 需求: 组织结构分析

Agent 能够评估组织结构健康度。

#### 场景: 组织层级分析

- **当** 分析汇报关系数据
- **则** 计算：
  - 组织层级深度
  - 平均管理幅度
  - 单点依赖岗位（只有一个人的关键岗位）
  - 管理者占比
- **且** 识别组织结构问题

#### 场景: 跨度与深度优化

- **当** 检测到结构问题
  - 层级过深（> 8 层）
  - 管理幅度过小（< 3 人）
  - 管理幅度过大（> 15 人）
- **则** 生成结构优化建议
- **且** 模拟优化后的组织架构

---

### 需求: 人口结构分析

Agent 能够分析员工人口统计特征。

#### 场景: 司龄结构分析

- **当** 分析员工司龄数据
- **则** 生成分布报告：
  - 司龄分布（0-1年、1-3年、3-5年、5-10年、10年+）
  - 平均司龄
  - 新员工占比趋势
  - 老员工占比趋势
- **且** 评估组织活力（新老搭配是否健康）

#### 场景: 年龄结构分析

- **当** 分析员工年龄数据
- **则** 生成分布报告：
  - 年龄分布
  - 平均年龄
  - 各年龄段占比趋势
- **且** 识别年龄断层风险
- **且** 预测未来 5 年退休人数

#### 场景: 多样性分析

- **当** 分析员工多样性数据
- **则** 计算：
  - 性别比例
  - 学历分布
  - 地域分布
  - 各维度在不同层级的分布
- **且** 对比行业基准

---

### 需求: 敬业度分析

Agent 能够分析员工敬业度。

#### 场景: 敬业度评估

- **当** 有员工调研数据
- **则** 分析敬业度：
  - 整体敬业度得分
  - 各维度得分（工作满意度、组织认同、发展机会等）
  - 按部门/职级的敬业度分布
  - 与历史对比趋势
- **且** 识别敬业度低的部门/群体

#### 场景: 敬业度驱动因素分析

- **当** 分析敬业度数据
- **则** 识别关键驱动因素：
  - 哪些因素与高敬业度正相关
  - 哪些因素与低敬业度负相关
  - 不同群体的差异化驱动因素
- **且** 生成提升敬业度的建议

---

### 需求: 组织健康综合评分

Agent 能够生成组织健康综合评估。

#### 场景: 计算组织健康度

- **当** 完成各维度分析
- **则** 生成组织健康度评分：
  - 人效维度得分
  - 编制维度得分
  - 结构维度得分
  - 人口维度得分
  - 敬业度维度得分
  - 综合健康度得分
- **且** 标记需重点关注的维度
- **且** 与历史/行业对标
