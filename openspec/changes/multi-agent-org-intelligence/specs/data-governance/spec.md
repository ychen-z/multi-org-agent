# 数据治理 Agent 规格

## Overview

数据治理 Agent 是整个系统的底座，负责从各种数据源接收原始 HR 数据，进行清洗、转换、验证，并统一数据口径，确保下游 Agent 使用的数据是干净、一致、可信的。

---

## 新增需求

### 需求: 数据清洗

Agent 能够自动识别并处理常见的数据质量问题。

#### 场景: 处理缺失值

- **当** 接收到包含缺失值的员工记录
- **则** 根据字段类型应用适当的填充策略
  - 数值型字段：使用中位数或均值填充
  - 类别型字段：使用众数或标记为"未知"
  - 日期型字段：标记为需人工审核
- **且** 记录所有填充操作到数据血缘日志

#### 场景: 处理重复记录

- **当** 检测到多条记录具有相同的员工唯一标识（工号）
- **则** 根据时间戳保留最新记录
- **且** 将重复记录移至历史表
- **且** 生成重复记录报告

#### 场景: 处理异常值

- **当** 检测到数值字段的值超出合理范围（如薪资<0，年龄>100）
- **则** 标记该记录为异常
- **且** 不将其纳入统计分析
- **且** 生成异常值报告供人工审核

---

### 需求: 口径统一

Agent 能够将来自不同系统的数据统一为一致的格式和标准。

#### 场景: 部门名称标准化

- **当** 接收到部门名称（如"研发部"、"研发中心"、"R&D"）
- **则** 映射到标准部门编码和名称
- **且** 保留原始名称作为别名

#### 场景: 职级体系统一

- **当** 接收到不同系统的职级表示（P6、高级工程师、Senior）
- **则** 转换为统一的职级代码和层级
- **且** 建立职级映射关系表

#### 场景: 日期格式统一

- **当** 接收到多种日期格式（2024-01-01、01/01/2024、1-Jan-24）
- **则** 统一转换为 ISO 8601 格式（YYYY-MM-DD）
- **且** 保存原始格式用于追溯

---

### 需求: 数据质量评估

Agent 能够对数据集进行质量评分和报告。

#### 场景: 生成数据质量报告

- **当** 完成一批数据的治理处理
- **则** 生成数据质量评分报告，包含：
  - 完整性评分（非空字段比例）
  - 一致性评分（符合标准格式比例）
  - 准确性评分（通过业务规则校验比例）
  - 及时性评分（数据更新及时程度）
- **且** 提供问题数据的详细清单

#### 场景: 数据血缘追踪

- **当** 任何数据字段被修改或转换
- **则** 记录完整的变更日志
  - 原始值
  - 新值
  - 变更原因
  - 变更时间
  - 变更规则

---

### 需求: 模拟数据生成

Agent 能够生成符合真实分布特征的大规模模拟 HR 数据。

#### 场景: 生成 500 万员工数据

- **当** 收到生成模拟数据的请求
- **则** 生成包含以下维度的员工数据：
  - 基础信息：工号、姓名、性别、年龄、学历、司龄
  - 组织信息：部门、职级、职位、汇报关系
  - 薪酬信息：基本工资、绩效奖金、总包
  - 绩效信息：年度绩效评级、OKR 完成度
  - 行为数据：考勤、培训、项目参与
- **且** 数据分布符合真实企业特征（如正态分布、幂律分布）
- **且** 保持数据间的逻辑一致性（如经理职级 > 下属职级）

#### 场景: 支持自定义数据规模

- **当** 指定生成数据的规模参数
- **则** 按比例生成各类数据
- **且** 保持数据关系的完整性

---

### 需求: 数据导入

Agent 支持多种格式的数据导入。

#### 场景: CSV 文件导入

- **当** 上传 CSV 格式的 HR 数据文件
- **则** 自动检测编码和分隔符
- **且** 解析表头并映射到标准字段
- **且** 执行数据清洗流程

#### 场景: Excel 文件导入

- **当** 上传 Excel 文件（.xlsx/.xls）
- **则** 支持多 Sheet 导入
- **且** 处理合并单元格和公式值
- **且** 执行数据清洗流程

#### 场景: API 数据同步

- **当** 配置外部系统 API 连接
- **则** 按计划定时拉取数据
- **且** 支持增量同步
- **且** 处理 API 分页和限流
