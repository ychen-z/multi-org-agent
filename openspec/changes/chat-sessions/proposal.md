# Chat Sessions 功能

## 概述

为 AI 对话页面添加多会话管理功能，支持对话列表、会话切换、建议问题填充等交互增强。

## 目标

1. 支持多个独立对话会话，可切换和回顾
2. 建议问题点击后填充到输入框，可编辑后发送
3. 会话按时间分组显示（今天/昨天/更早）
4. 自动清理空会话和限制会话数量

## 范围

- **存储**: LocalStorage 多会话
- **UI 布局**: 左侧会话列表 (240px) + 右侧对话区
- **会话标题**: 自动截取第一条消息前 20 字
- **会话管理**: 直接删除（无确认弹窗）
- **空会话清理**: 切换时自动清理无消息的空会话
- **会话数量**: 最多 20 个，超出删除最老的

## 非目标

- 后端持久化存储
- 用户认证
- 会话搜索/筛选
- 会话重命名
